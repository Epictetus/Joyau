Joyau by Kode

A ruby interpreter for PSP, with some extension so that it could do things such as image loading, ...

Lua is a popular programming language on the PSP, but Ruby is oriented object, and some people prefer it to Lua, even if it is slower. And the lib ruby has been ported two years before Joyau have started. However, there hasn't been that much project like it, and no one seems to have got as much features as Joyau. Now, Joyau aims to provied a high level interface, easy to use in a game for PSP.

Dependencies

You should of course have the psp toolchain installed, and the pspsdk too. The following library, compiled for the PSP, are needed to: devslib, OpenAL, libpng, zlib, libvorvis, libogg, and the libruby.

Install

If you don't want to compile your own EBOOT, use the provieded one, which can be downloaded in the Download section. After having the dependencies installed, clone the repository git://github.com/Kode/Joyau.git Then, go into the Joyau/Joyau folder, and execute "make". You'll get an EBOOT.PBP file. The eboot should be set in a subfolder of PSP/GAME/. You should also set the script.rb file in that folder.
